## Installation

1. Clone the repository:

```bash
git clone https://github.com/4lando/landofile-editor.git
cd landofile-editor
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
lando start
```

4. Open your browser and navigate to `https://landofile-editor.lndo.site`.
