Usage Instructions

The editor validates your Landofile configuration against the official schema.

Key Features:

  • YAML syntax highlighting
  • Real-time validation
  • Auto-completion
  • Error highlighting

Common Fields:

  • name: Project name
  • recipe: Base configuration
  • config: Recipe settings
  • services: Custom services

Validation powered by the Lando Schema Specification
